# BTT500

新加坡基本交通理论 (Basic Theory Test) 考试练习 Android App。

## 功能

- **随机抽题练习**: 每次练习 50 道题，完成后显示得分（90% 为通过线）
- **智能优先级**: 优先抽取之前答错的题目和练习次数最少的题目
- **历史记录浏览**: 查看所有题目的练习次数和错误次数
- **多种排序**: 支持按默认顺序、练习次数最少、错误次数最多排序
- **题目详情**: 查看每道题的全部答题记录（全部记录 / 最近十次）
- **中英双语**: 题目和选项同时显示中文和英文

## 题库

包含 610 道 BTT 考试题目，涵盖：
- 交通标志与标线
- 变道与转弯
- 防御性驾驶
- 超车规则
- 停车规则
- 速度与限速
- 酒驾与药物
- 行人与弱势道路使用者
- 交通信号灯
- 高速公路驾驶
- 车辆保养与检查
- 天气与能见度
- 车辆灯光使用
- 驾照与证件
- 让行与优先权
- 事故与紧急情况

## 构建

```bash
./gradlew assembleDebug
```

APK 输出路径: `app/build/outputs/apk/debug/app-debug.apk`

## 技术栈

- Java 17
- Android SDK 34 (minSdk 26)
- Room Database (本地存储答题记录)
- Material Design Components
- Gson (JSON 解析)
